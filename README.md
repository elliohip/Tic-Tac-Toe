# Tic-Tac-Toe
Simple tic tac toe game

There are three difficulties one can select, easy, medium, and hard.

Easy : almost no use

Medium : Random move

Hard : unbeatable tic tac toe


# unbeatable AI

Array implementation of minimax algorithm.
